# Join requests (main register)

**When?** A **satellite** sent a join request with the **6-digit join code** – you **accept** or **reject** it.

## Where?

- **Home**: **Join requests** tile (badge when pending).
- **Status bar**: **X join request(s) pending** (when any exist).

## Flow

1. Open **Join requests**.
2. Each row shows **name**, **register id**, and **sync URL** (if present).
3. **Accept** or **Reject**.

The list refreshes in the background.

## Before join (satellite)

The satellite needs:

- valid **main register URL**,
- **own sync URL**,
- **join code** from main [Settings](handbuch://einstellungen).

See [Settings](handbuch://einstellungen) and [First run & roles](handbuch://erststart-und-rollen).

## Common issues

- **No requests** → main **server** started? Satellite ran **Join network**?
- **Reject** → satellite may need to join again.

See also: [Technical / administration](handbuch://technik)
