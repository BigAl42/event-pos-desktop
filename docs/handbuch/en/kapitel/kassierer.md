# Cashier (operation)

**When?** You work the register, void receipts, or need the main clicks—without network details.

## In this section

| Topic | Content |
| --- | --- |
| [Register](handbuch://kasse) | Customer receipts, quick entry, keyboard |
| [Void](handbuch://storno) | Void one line or a full receipt |
| [Settlement](handbuch://abrechnung) | Totals per merchant, PDFs, (main) close wizard |

## Typical flow

1. **Home** → open **Register**.
2. Check staff, enter lines → **Complete customer receipt**.
3. Errors: use **Void**.
4. End of day (per role): **Settlement** or follow main-register instructions.
