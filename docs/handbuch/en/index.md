# Register system handbook

Welcome to the user guide. This handbook ships with the app version and is available **offline**.

## Two entry points

- **[Cashier (operation)](handbuch://kassierer)** – register, voids, settlement from a cashier perspective.
- **[Technical / administration](handbuch://technik)** – setup, network, sync, merchants, emergency mode.

## Quick start (cashier)

1. **Home** → **Register** → enter lines → **Complete customer receipt**.
2. Wrong entry: **Void** → pick receipt → void line or whole receipt.
3. **Settlement**: check totals, create **PDF** or **Create all PDFs** as needed.

## Handbook in the app

- **Entry**: **Handbook** tile on the home screen, link in **Settings**, **Help** in the **status bar** (footer).
- **PDF**: In the handbook view use **Current chapter as PDF** or **Full handbook as PDF**.

## Roles (short)

- **Main register (master)**: server for satellite registers, merchant admin, join requests, **Close billing cycle** (wizard).
- **Satellite register (slave)**: connect to main, post sales, request closeout before leaving; merchant overview is **read-only**.
