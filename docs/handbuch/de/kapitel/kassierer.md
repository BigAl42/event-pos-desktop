# Kassierer (Bedienung)

**Wann brauche ich das?** Du arbeitest an der Kasse, stornierst Belege oder brauchst schnell die wichtigsten Klicks – ohne Netzwerk-Details.

## In diesem Bereich

| Thema | Inhalt |
| --- | --- |
| [Kasse](handbuch://kasse) | Kundenabrechnung erfassen, Schnelleingabe, Tastatur |
| [Storno](handbuch://storno) | Einzelposition oder ganzen Beleg stornieren |
| [Abrechnung](handbuch://abrechnung) | Summen pro Händler, PDFs, (Hauptkasse) Abschluss-Wizard |

## Typischer Ablauf

1. **Startseite** → **Kasse** öffnen.
2. Besetzung prüfen, Positionen eintragen → **Kundenabrechnung abschließen**.
3. Bei Fehlern: **Storno** nutzen.
4. Am Ende des Tages (je nach Rolle): **Abrechnung** oder die Anweisungen der Hauptkasse befolgen.

## Technik & Einrichtung

Alles zu Hauptkasse/Nebenkasse, Netzwerk und Einstellungen steht unter [Technik / Administration](handbuch://technik).
