# Einstellungen

Die Einstellungen sind in einklappbaren Bereichen (Akkordeon) organisiert.

## Diese Kasse

- Kassenname, Rolle (Hauptkasse/Nebenkasse).
- Besetzung: Person 1 und Person 2 bearbeiten.

## Netzwerk (Hauptkasse)

- Server-Port und eigene Sync-URL.
- Join-Token generieren (6-stelliger Code für Slaves).
- Server starten, danach „Sync zu Peers starten“.

## Netzwerk (Nebenkasse)

- Hauptkasse suchen oder URL eintragen.
- Eigene Sync-URL, Join-Code, „Netz beitreten“.
- Sync starten, ggf. Reset-Anfrage, Closeout anfragen, „Abmelden & entkoppeln“.

## Abrechnungsläufe

- Liste der Läufe, aktiver Lauf gekennzeichnet.
- Neuen Abrechnungslauf starten (löscht alle Buchungen/Kundenabrechnungen).

## Notfallmodus

- Export/Import von Bewegungsdaten als Excel oder CSV zur manuellen Zusammenführung auf anderer Kasse.
