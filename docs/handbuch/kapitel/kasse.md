# Kasse

Über das Tile **Kasse** auf der Startseite gelangst du zur Kundenabrechnung.

## Ablauf

1. Besetzung anzeigen bzw. ändern (Person 1 / Person 2).
2. Positionen erfassen: Händlernummer, Betrag, optional Bezeichnung.
3. Belegnummer wird automatisch vergeben (Format: Prefix-Jahr-NNN).

## Hinweise

- Jede Kundenabrechnung kann mehrere Positionen haben.
- Stornos sind über die **Storno**-Ansicht möglich.
